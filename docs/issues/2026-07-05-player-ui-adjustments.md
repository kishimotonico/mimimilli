# 再生UIのバー復元とポップアップ導線調整

Status: done

## 背景

`fbb53a97491183ac498f97b20334cf1c1e56b851` で画面下の再生バーに時間表示つきの横長シーク行を追加したが、DLsite Play を参考にした下辺だけのシークUIのほうがアプリの常駐バーとして自然だった。

ポップアップ再生UIでは、「再生中の作品を表示」が下部に独立していて、上部のサイズ変更ボタンが左寄せだったため、操作のまとまりが弱かった。

## 対応

- 画面下バーのシークUIを、親コミットの下辺シークバー形式に戻した
- バー内の経過時間・総時間表示とスライダー風のつまみを撤去した
- ポップアップ左上に「再生中の作品を表示」ボタンを移動した
- ポップアップ右上に「バーへ戻る」「全画面プレイヤー」ボタンをまとめた
- ポップアップ下部の「再生中の作品を表示」行を削除した
- 作品一覧とファイル一覧の再生中アイコンを、実際の再生状態に連動させた。一時停止中は対象表示を残しつつアニメーションを止める
- 作品一覧右端の相対追加日（例: `3 週間前`）を削除した
