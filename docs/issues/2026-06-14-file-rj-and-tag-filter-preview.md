# ファイルRJ表示とタグ絞り込みプレビューの修正

Status: done

## 目的

- ファイルモードの登録作品フォルダーで RJ コードが二重に見える表示を解消する
- タグ絞り込み結果に未選択向けの案内と「概要」見出しが残る矛盾を解消する
- パンくずと物理パスは実フォルダー名のまま維持する

## 設計

- 登録作品フォルダーの表示は純関数で `badge` と `name` に分離する
- RJ 作品はコード全体をバッジにし、実フォルダー名が同一コードと区切りで始まる場合だけ接頭辞を除去する
- 非 RJ 作品と、安全にタイトルを切り出せないフォルダー名は「作品」バッジと従来の名前を維持する
- axis-landing の文言は純関数で導出し、タグ選択済みの場合だけ「絞り込み結果」として案内を非表示にする
- タグ絞り込み適用状態は `LibraryView` の `selectedTags` から導出する
